# Hunter X Hunter New Tab Page

![Screenshot of Hunter X Hunter New Tab Page](screenshots/preview.png)

A custom new tab page for your browser, transforming your new tab experience with a vibrant **Hunter X Hunter theme**. This extension is designed to provide a quick and immersive start to your Browse session, featuring essential tools wrapped in the adventurous spirit of the Hunter Association!

## ‚ú® Features

* **Hunter X Hunter Themed Clock:** A stylish clock display integrated seamlessly with the HxH aesthetic.
* **Integrated Google Search:** Quickly search the web directly from your new tab.
* **Dynamic Weather Display:** Get current weather conditions for your location (requires location access).
* **Curated Manga Recommendations:** Discover or revisit popular manga titles, including Hunter X Hunter and others.
* **Character Display (Optional/Planned):** (e.g., Gon, Killua, Kurapika, Leorio) - if you want to add images or details here.
* **Multiple Themes (Planned):** Future support for various Hunter X Hunter-inspired themes to customize your look.

## üöÄ Installation

This project is primarily designed to be used as a **Custom New Tab URL** with extensions like [New Tab Override](https://addons.mozilla.org/firefox/addon/new-tab-override/) (for Firefox) or similar functionality in Chrome.

### For Firefox Users (via New Tab Override Add-on):

1.  Install the [New Tab Override](https://addons.mozilla.org/firefox/addon/new-tab-override/) add-on for Firefox.
2.  Go to the **Add-on settings** for "New Tab Override".
3.  Set the custom URL to:
    ```
    [https://rhevaldi.github.io/HXH-NewTab/](https://rhevaldi.github.io/HXH-NewTab/)
    ```

### For Chrome Users (Manual Installation as Unpacked Extension):

*If you prefer to install it as a standalone Chrome extension:*

1.  **Clone or Download:** Clone this repository or download the ZIP file and extract it to a local folder (e.g., `C:\xamppp\htdocs\HXH-NewTab\`).
2.  **Open Chrome Extensions:** Go to `chrome://extensions` in your Chrome browser.
3.  **Enable Developer Mode:** Toggle on "Developer mode" in the top-right corner.
4.  **Load Unpacked:** Click the "Load unpacked" button.
5.  **Select Folder:** Navigate to and select the extracted `HXH-NewTab` folder (which should contain `manifest.json`).
